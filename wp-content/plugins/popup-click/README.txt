== Description ==

**OVERVIEW**

Pop-Up CC Click plugin brings unlimited on click pop-ups with professional templates. The pop-ups are responsive and work with any WordPress theme.

Display your videos in elegant pop-ups and decide when the visitors see them - choose the page element that will trigger the pop-up when clicked.

== Installation ==


= Using The WordPress Dashboard =


1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'pop-up-click'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard
5. Go to PopUp in the left menu to manage the pop-ups


= Uploading in WordPress Dashboard =


1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `pop-up-click` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard
6. Go to PopUp in the left menu to manage the pop-ups


= Using FTP =


1. Download 'pop-up-click'
2. Extract the 'pop-up-click' directory to your computer
3. Upload the 'pop-up-click' directory to the '/wp-content/plugins/' directory
4. Activate the plugin in the Plugin dashboard
5. Go to PopUp in the left menu to manage the pop-ups
